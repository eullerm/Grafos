Trabalho de APA

Existem duas formas de fazer o teste do programa.

Primeiro, pelo executavel:
	1 - Acessar a pasta dist;
	2 - Executar o .exe com nome "trabalho de APA";
	3 - Clicar no local onde tem um exemplo de input;
	4 - Os inputs devem seguir o exemplo da caixa de texto, separados por traços e ponto-e-vírgula;
	5 - Clicar no botão "Criar grafo".


Segundo, pelo VSCode:
	1 - Precisa ter o python e pygame instalados.
	2 - Abrir o main.py
	3 - Apertar 'F5' e selecionar 'Python File'
	4 - Clicar no local onde tem um exemplo de input;
	5 - Os inputs devem seguir o exemplo da caixa de texto, separados por traços e ponto-e-vírgula;
	6 - Clicar no botão "Criar grafo". 


	Para editar os pesos das arestas deve-se clicar no valor numérico do peso dela e para editar o peso dos vértices deve-se clicar no próprio vértice.
	O botão "Grafo" exibe o grafo de forma circular.
	O botão "Matriz de adjacência" exibe uma matriz de 0’s e PESOS do grafo atual.
	O botão "Matriz de incicência" exibe uma matriz com n vértices e m arestas do grafo atual.
	O botão "Voltar" permite que se adicione mais conexões ou apague as existentes, entretanto o peso dos vértices e arestas são resetados.
	
	**Para instalar o python basta acessar o site https://www.python.org/
	**Para instalar o pygame basta digitar no terminal: pip install pygame
